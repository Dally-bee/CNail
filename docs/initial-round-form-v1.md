## 3. 具体业务问题

中国中小跨境卖家能够快速获取视觉素材，却往往缺少判断美国文化语境、趋势证据和具体使用场景的能力。面对穿戴甲等快反品类，卖家很难同时回答：某个文化符号在美国消费者语境中意味着什么、适合哪类场景、哪些表达可能造成文化误读，以及如何把趋势线索转成可进入选款和打样的商品方案。现有趋势工具常停留在热度展示，设计工具又容易直接生成“好看但无证据、无边界、不可执行”的结果。CultureNail 首版聚焦 2026 年 Q4、美国市场、Amazon US 穿戴甲和骷髅符号，解决从公开趋势证据到可测试上新概念之间的断层。

## 4. 方案名称

CultureNail——面向中国中小跨境卖家的美国文化趋势上新智能体

## 5. 一句话定义

CultureNail 将带来源和时间的公开趋势证据，转换为可进入人工选款、打样和 Amazon US 上架测试的穿戴甲商品概念套装。

## 6. 3–5 个核心功能

1. 机会雷达：整合版本化公开证据，保留来源 URL、采集时间、市场、指标、局限和置信度，形成可测试的上新机会假设。
2. 文化语境分析：区分万圣节派对、日常哥特和亡灵节相关纪念语境，为同一骷髅符号设置不同的适用表达、避免项和人工复核闸门。
3. 商品概念套装：生成目标人群、使用场景、配色、图案、甲型、文化护栏和小批测试计划，供人工选款与供应链打样，不冒充生产级设计稿。
4. 双边合成研究：通过 12 名美国女性消费者模拟角色和 5 名中国跨境卖家模拟角色进行 `AI_SIMULATION` 情景压力测试，暴露偏好、执行和文化风险，不替代真实访谈或市场验证。
5. 可追溯 Listing 草稿：生成 Amazon US 标题、要点、描述和搜索词假设，每项 AI 结论关联证据 ID，并保留人工合规复核提示。

## 7. 方案亮点

- 证据可追溯：不是直接“猜趋势”，而是让每个 AI 结论回到带来源、时间和局限说明的公开证据快照；Google Trends 如后续使用，只解释为特定范围内的相对关注度，不包装为销量。
- 文化边界内建：同一符号跨语境时强制分流。亡灵节是与逝者纪念、家庭和具体文化传统相关的独立语境，不等同于恐怖素材或“墨西哥万圣节”。
- 双边可执行性：同时检查美国消费者场景假设与中国卖家的选款、打样、成本、Listing 和小批测试约束。
- 演示稳定：固定证据和 mock 响应均可本地回退；没有赛事算力、网络或外部模型时，五屏主演示仍可完整走通。

本方案不宣称预测爆款、销量或真实购买意向，也不把合成反馈表述为真实用户验证或市场比例。

## 8. 技术方案说明

首版采用 Next.js + TypeScript 构建单页 Web Demo，通过服务端 API Route 统一处理文化分析、概念生成、合成评价和 Listing 生成。趋势证据、文化语境、商品概念及 12+5 模拟角色使用版本化 JSON 保存，第一版不引入数据库。

文本生成通过统一 `ModelProvider` 接口连接确定性 mock Provider 与百炼 OpenAI 兼容适配器。当前已经完成并验证无密钥 mock 路线；百炼适配器只允许服务端从环境变量读取配置，并设有超时、输出校验和同契约 mock 回退，但尚未进行真实百炼 API 测试。万相仅预留禁用的服务端接口，不提交图像任务。所有外部数据或模型失败时，Demo 都可使用本地证据快照和 mock 响应完成主演示。
